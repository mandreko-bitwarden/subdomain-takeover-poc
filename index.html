What is this?

This page serves as a proof-of-concept for subdomain takeover scenarios. It demonstrates that a GitHub Pages site can be successfully configured and hosted at this domain.

Security Note

Subdomain takeovers occur when a subdomain (e.g., subdomain.example.com) points to a service that is no longer in use. This allows an attacker to claim the subdomain and host content.
